Tweet Sentiment Analyzer.

In this program, you will build a rough sentiment analyzer for tweets from complete scratch

1. Search for two dictionaries online. One with positive words, and one with negative words. Save these into separate files within the data folder.

2. Retrieve the Sentiment Analysis dataset.

cd data

wget http://thinknook.com/wp-content/uploads/2012/09/Sentiment-Analysis-Dataset.zip

unzip Sentiment-Analysis-Dataset.zip

3. Think of a way to check how likely this tweet is positive or negative and write a program for it. Iterate on it and check if it works!

4. Save your code within ha_01_python102.py and finish by next class.
