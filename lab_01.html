lab 1 If statement

Goals

  • To understand the anatomy of an if statement
  • Write your own simple if statement, and nested if statements

Simple if statement

Try to predict what this file will output

simple_if.py

a = 10
b = 5

if a > b:
  print("a is greater than b")
elif a == b:
  print("a is equal to b")
else:
  print("a is less than b")

Execute:

python simple_if.py

Output:

$ python simple_if.py


a is greater than b

Nested if statement

Try to predict what this file will output

nested_if.py

a = 10
b = 5

if a > b:
  print("a is greater than b")
  if a + b == 15:
    print("a + b is 15")
  else:
    print("a + b is not 15")
elif a == b:
  print("a is equal to b")
else:
  print("a is less than b")

Execute:

python nested_if.py

Output:

$ python nested_if.py


a is greater than b

a + b is 15
