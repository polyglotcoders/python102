lab 2 Iterating

Goals

  • Understand for loops / nested for loops

Overview

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string)

Here are a couple of for loops in action

Iterating by item with a for loop

by_item.py

fruits = ["apple", "banana", "cherry"]

for x in fruits:
  print(x)

Execute:

python by_item.py

Output:

$ python by_item.py


apple

banana

cherry

Iterate via index with a for loop

Another useful way to iterate over items is via their index. Say that we only wanted to iterate over the first 2 words

by_range.py

fruits = ["apple", "banana", "cherry"]

for i in range(0,2):
  print(fruits[i])

Execute:

python by_range.py

Output:

$ python by_range.py


apple

banana

Now by_range.py is equivalent to

clone_list.py

fruits = ["apple", "banana", "cherry"]
first_two = fruits[:2]

for x in first_two:
  print(x)

Execute:

python clone_list.py

Output:

$ python clone_list.py


apple

banana

However, by_range.py is more ideal because it takes up less memory as it doesn’t have to allocate more space for a cloned list

Advanced: List comprehension

To create a new list in one line, use list comprehension

one_line.py

fruits = ["apple", "banana", "cherry"]

# Via item

# List iterator
uppercased_list = [x.upper() for x in fruits]

# Which is equivalent to this for loop
uppercased_list = []
for x in fruits:
  uppercased_list.append(x.upper())

# Via index

# List iterator
uppercased_list_via_index = [fruits[i].upper() for i in range(0, 2)]

# For loop
uppercased_list_via_index = []
for i in range(0,2):
  uppercased_list_via_index.append(fruits[i].upper())
