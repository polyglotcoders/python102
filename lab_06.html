Goals

  • To understand the anatomy of a function

Code can get long and repetitive!

Here’s a function that prints out a greeting and a name

no_function.py

names = ['Tim', 'Bea', 'Queenie']

for name in names:
  if name == 'Tim':
    print('Hello Tim')
  elif name == 'Bea':
    print('Hello Bea')
  elif name == 'Queenie':
    print('Hello Queenie')
  else:
    print('Hello')

Execute:

python no_function.py

Output:

$ python no_function.py


Hello Tim

Hello Bea

Hello Queenie

Imagine if you had to do this for 10000 names. Think about how long and repetitive this code will be!

Functions

Functions are a set of instructions that you may want to call repeatedly, or perhaps because of their complexity, you want to hide.

Each function carries out a specific task. For example

with_function.py

names = ['Tim', 'Bea', 'Queenie']

def say_hello(name):
  print('Hello {}'.format(name))


for name in names:
  say_hello(name)

Execute:

python with_function.py

Output:

$ python with_function.py


Hello Tim

Hello Bea

Hello Queenie

The return statement

Add return statements in order to return a value, which can then be stored in a variable

with_function_return.py

names = ['Tim', 'Bea', 'Queenie']

def say_hello(name):
  return 'Hello {}'.format(name)


for name in names:
  # Since a value is returned from this function, we can store it in a variable
  greeting = say_hello(name)

  # Now we can print that variable value
  print(greeting)

Execute:

python with_function_return.py

Output:

$ python with_function_return.py


Hello Tim

Hello Bea

Hello Queenie
